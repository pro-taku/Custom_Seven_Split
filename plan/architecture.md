### 데이터 흐름 (Core Logic)

```
[Price Watcher] ──▶ [KIS API] ──▶ [현재가 수신]
       │
       ▼
[TradeEngine]
   1. StockStrategy 확인 (매수 갭, 익절 목표)
   2. VirtualBalance 확인 (현재 1~7번 계좌 보유 현황)
   3. 판단:
      ├─ 매수: 이전 차수 대비 -N% 하락 시 ──▶ [매수 주문] ──▶ [VirtualBalance 추가]
      └─ 매도: 개별 차수 목표 +N% 도달 시 ──▶ [매도 주문] ──▶ [VirtualBalance 삭제]
                                                         ──▶ [TradeHistory 기록]
```
